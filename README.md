# 干燥综合征健康管理 H5应用

## 在线访问

**网址**：https://fang618719.github.io/sjogren-health

> 如遇更新不生效，请在网址后加版本参数，如 `?v=16`，或使用无痕模式打开。

## 手机使用方法

1. 手机浏览器打开上述网址
2. 输入用户名和密码登录
3. （推荐）点击浏览器菜单 → "添加到主屏幕"，像APP一样使用

## 功能概览

### 患者管理
- 多患者管理：添加、搜索、筛选（全部/未开始/进行中/已完成/已剔除）
- 纳入排除筛查：添加患者时进行纳排标准核查
- 剔除与恢复：可剔除患者并填写原因，支持恢复
- 纳排复核：可重新审核患者纳排状态

### 数据采集
- 13个数据采集模块（基本信息、症状、检查、抗体、器官受累、ESSDAI、中医四诊、中医证候、治疗、SF-36、心理、超声、随访）
- **自动保存**：输入变化时自动保存，无需手动点击
- 进度追踪：模块完成度 + 关键字段完成率
- 缺失清单：一键查看待填写的关键字段

### 数据导出
- **单患者导出**：导出单个患者CSV
- **批量导出**：支持长表/宽表格式
- **导出选项**：可选是否包含已剔除患者、是否脱敏处理

### 数据备份
- **导出备份**：一键导出JSON备份文件（含所有患者数据）
- **导入恢复**：支持从备份文件恢复数据
- **自动提醒**：每日首次登录提醒备份

## 注意事项

- 数据存储在手机本地（localStorage），清除浏览器数据会丢失记录
- 请定期使用"备份/恢复"功能导出备份文件
- 不同设备/浏览器数据不互通，建议固定使用同一浏览器
