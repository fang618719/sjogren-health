# 干燥综合征健康管理 H5应用

## 使用方法

### 方法1：本地直接使用
双击打开 `index.html` 即可在浏览器中使用

### 方法2：部署到GitHub Pages（推荐）
1. 注册GitHub账号：https://github.com
2. 创建新仓库，命名如 `sjogren-health`
3. 上传 `sjogren_h5` 文件夹内所有文件
4. 进入仓库 Settings → Pages → Source选择main分支
5. 等待几分钟后访问：`https://你的用户名.github.io/sjogren-health`
6. 将链接发给对方，手机打开即可使用

### 方法3：添加到手机桌面
1. 手机浏览器打开链接
2. 点击浏览器菜单 → "添加到主屏幕"
3. 像APP一样从桌面打开使用

## 功能说明
- 13个数据采集模块
- 数据存储在手机本地（localStorage）
- 支持导出CSV文件
- 4位数字密码保护

## 注意事项
- 清除浏览器数据会丢失记录，请定期导出备份
- 首次使用设置的密码请牢记
